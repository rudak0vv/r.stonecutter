# ⛏ r.stonecutter

Плагин для **Paper 1.21.1**, расширяющий функциональность камнереза дополнительными QoL-рецептами.

## Возможности

### 🪵 Дерево
**Из досок** (любых пород) в камнерезе:
- 2× Плита
- 1× Ступенька
- 2× Палки

**Из брёвен** (включая обтёсанные, всех пород) в камнерезе:
- 4× Доски
- 4× Кнопки
- 2× Забор
- 1× Калитка
- 2× Нажимная плита
- 1× Люк
- 8× Палки
- 1× Обтёсанная древесина

> Рецепты динамические — тип дерева сохраняется (дубовое бревно → дубовые изделия и т.д.)

---

### 🟧 Медь
**Из медного блока** (любая степень окисления, вощёный/невощёный):
- 9× Медный слиток
- 4× Точёный медный блок
- 4× Медная дверь
- 4× Медная решётка
- 3× Медный люк
- 4× Резной медный блок
- 8× Резная медная плита
- 4× Резные медные ступеньки
- 3× Громоотвод

**Из ступеньки или плиты** (медь):
- 1× Медный слиток

**Снять воск** — вощёный блок/плита/ступенька → такой же без воска (степень окисления сохраняется)

**Убрать окисление** — oxidized → weathered → exposed → copper (в обратную сторону)

> Из обычной (необокислённой) меди нельзя сделать окислённую — это нелогично.

---

### 🪟 Стекло
- Любое цветное стекло → бесцветное стекло
- Любая цветная стеклянная панель → бесцветная панель

---

### 🧱 Терракота
- Любая глазурованная терракота → терракота
- Терракота → белая терракота (и другие цвета)

---

### 💎 Аметист
- Кластер аметиста → 4× осколок
- Блок аметиста → 4× осколок
- Бутон аметиста → 4× осколок

---

### 🪨 Камень
- Камень → булыжник
- Камень → 2× плита из булыжника
- Камень → 1× ступенька из булыжника
- Камень → 1× ограда из булыжника

---

### 🖼 Картины
Любую картину (`minecraft:painting`) можно переработать в камнерезе в конкретный вариант (Alban, Kebab, Wanderer, Pointer и т.д.) — все 50+ вариантов доступны.

## Команды

| Команда | Описание | Права |
|---|---|---|
| `/rsc-reload` | Перезагрузить рецепты из config.yml | `rstonecutter.reload` |

## Конфиг (`config.yml`)

Все рецепты хранятся в `config.yml` и полностью настраиваемы.

### Структура рецепта

```yaml
recipes:
  my_recipe:
    enabled: true          # включить/выключить рецепт
    input: "minecraft:stone"   # входной предмет (или tag: вместо input)
    output: "minecraft:cobblestone"
    amount: 1              # кол-во выходных предметов
    input-count: 1         # сколько входных предметов нужно (по умолчанию 1)
```

### Поля

| Поле | Описание |
|---|---|
| `enabled` | `true`/`false` — включить рецепт |
| `input` | Конкретный предмет на входе (`namespace:id`) |
| `tag` | Тег вместо конкретного предмета (напр. `minecraft:logs`) |
| `output` | Выходной предмет |
| `amount` | Количество выходных предметов |
| `input-count` | Сколько входных предметов нужно (по умолчанию 1) |
| `exclude_input` | Исключить конкретный предмет из тега |
| `dynamic_wood` | `true` — тип дерева определяется автоматически |
| `wood_target` | Цель для дерева: `planks`, `slab`, `stairs`, `button`, `fence`, `fence_gate`, `pressure_plate`, `trapdoor`, `stripped_wood` |
| `dynamic_copper` | `true` — степень окисления/воск определяется автоматически |
| `copper_target` | Цель для меди: `chiseled`, `door`, `grate`, `trapdoor`, `carved_slab`, `carved_stairs`, `dewax_block`, `dewax_slab`, `dewax_stairs` |
| `painting_variant` | Вариант картины: `minecraft:alban`, `minecraft:wanderer` и т.д. |

### Пример: добавить новый рецепт

```yaml
recipes:
  # Гравий → кремень
  gravel_to_flint:
    enabled: true
    input: "minecraft:gravel"
    output: "minecraft:flint"
    amount: 1

  # Отключить рецепт
  stone_to_cobblestone:
    enabled: false
    input: "minecraft:stone"
    output: "minecraft:cobblestone"
    amount: 1
```

## Установка

1. Скопировать `r.stonecutter-*.jar` в папку `plugins/`
2. Перезапустить сервер
3. Отредактировать `plugins/r.stonecutter/config.yml` при необходимости
4. Применить изменения: `/rsc-reload`

---

## Требования

- **Paper** 1.21.1+
- **Java** 17+

---

## Сборка

```bash
mvn package
```

JAR будет в `target/r.stonecutter-*.jar`.
